HOK Regl - Flutter project (prepared)

Contents:
- lib/: Dart source files (main.dart, data_manager.dart, notification_service.dart)
- pubspec.yaml
- assets/logo.png
- prepare.sh : run this in Gitpod or local machine to ensure platform folders and dependencies are ready

Quick start (Gitpod):
1. Open repository in Gitpod.
2. Run: ./prepare.sh
3. Then: flutter build apk --release
4. Download app-release.apk from build output.

Note: iOS builds require macOS and appropriate signing.
